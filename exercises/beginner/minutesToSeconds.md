# Minutes to Seconds

## EN

### Task
Write a program that convert `minutes` variable to seconds.

### Example
```
15 minutes = 900 seconts
4.5 minutes = 270 seconds
```

## HU

### Feladat
Írj egy programot, amely `munites` iteger típusú változót másodpercekké konvertálja.

### Példa
```
15 perc = 900 masodperc
4.5 perc = 270 masodperc
```
